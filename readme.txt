== Description ==

The FireDrum Email Marketing plugin allows you to quickly and easily add an opt-in form for your FireDrum Email Marketing list. Use shortcode [firedrum_marketing] in templates, posts or pages to display the opt-in form.

== Installation ==

This section describes how to install the plugin and get started using it.

= Version 2.8+ =
1. Unzip our archive and upload the entire `firdrum` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Firedrum API Key and let the plugin verify it.
4. Finally, go to Appearance->Widgets and drag the `FireDrum Widget` widget into one of your Widget Areas OR use shortcode <strong>[firedrum_marketing]</strong> to display it on any template, post or page.
5. And you are DONE!
